## Ringkasan Praktikum

- Membangun topologi jaringan sederhana (attacker, client, server) pada jaringan 192.168.132.0/24.
- Melakukan ARP spoofing untuk membuat attacker menjadi man-in-the-middle, kemudian mengamati sniffing dan session hijacking pada Telnet serta perbandingannya dengan SSH.
- Menguji beberapa serangan IP spoofing menggunakan `hping3` (ICMP flood/pod, TCP SYN flood ke port 22, UDP flood, dan land attack) lalu mengamati dampaknya terhadap resource server dan koneksi SSH.
